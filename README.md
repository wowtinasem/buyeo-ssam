# 교원AI 수업역량 강화 연수 성과 전시관 — 부여교육지원청

2026년 7월 14일(월)~16일(수) 진행된 **바이브코딩 교원연수**에 참여한 교원 15명이
자연어 대화만으로 직접 제작한 수업용 앱을 소개하는 온라인 전시관 페이지입니다.

- 파일: `index.html` (외부 라이브러리 없는 단일 HTML — 그대로 열거나 정적 호스팅에 올리면 동작)
- 디자인 시안: C3 — 남색 공공기관 톤 + 글래스 연꽃(궁남지 모티프) + 디지털 닷 글로브

## 페이지 구성

| 영역 | 내용 |
| ------ | ------ |
| 기관 표시줄 | 충청남도부여교육지원청 · 2026 교원역량강화 직무연수 |
| 히어로 | "교원AI 수업역량 강화 연수 성과 전시관" 아이브로우 + 키네틱 타이포 + 타이핑 프롬프트 + 연꽃/글로브 연출 |
| 필터 | 전체 / 초등부 / 중등부 |
| 갤러리 | 작품 카드 15개 (작품 번호 · 학교 · 교사명 · 부문 칩) |
| 도록형 모달 | 작품 주제 · 사용 방법 · 앱 바로가기 버튼 |
| 자동 전시 모드 | 40초 무조작 시 작품 카드가 6.5초 간격으로 자동 순환, 조작 시 즉시 해제 (행사장 키오스크용) |

## 작품 데이터 수정 방법

HTML 안 `const APPS = [ ... ]` 배열만 고치면 됩니다. 현재 15개는 **모두 예시 데이터**입니다.

| 필드 | 설명 |
| ------ | ------ |
| `school` | 학교 이름 |
| `teacher` | 선생님 성함 |
| `level` | `"초등"` 또는 `"중등"` (필터에 사용) |
| `title` | 앱(작품) 이름 |
| `topic` | 작품 주제 1~2문장 (교과·학년 병기 권장) |
| `usage` | 사용 방법 ①②③ — 줄바꿈은 `\n` |
| `url` | 앱 주소. 비우면(`""`) 버튼이 "앱 주소 준비 중"으로 표시 |
| `emoji` | 카드 이모지 1개 |
| `image` | 앱 캡처 이미지 주소(선택). 비우면 자동 색상 카드 |

## 구현 세부

- **애니메이션**: 히어로 키네틱 등장(1회) → 시머·부유·연꽃 개화 루프. `prefers-reduced-motion` 사용자에겐 모든 모션 정지 및 정적 표시.
- **디지털 글로브**: canvas에 피보나치 구 분포 점 720개를 그려 자전(1바퀴 약 65초). 탭이 숨겨지면 렌더링 정지.
- **접근성**: 카드가 `button` 요소, 모달은 `role="dialog"` + Esc 닫기 + 포커스 복원 + Tab 포커스 트랩.
